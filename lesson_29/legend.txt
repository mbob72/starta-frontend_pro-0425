spa - single page application
(при изменении страницы или отправке запросов сайт не перезагружается)

.gitignore - указываем файлы которые git должен игнарировать не добавлять в репозиторий
eslint.config.js - настройки линтера (отвечат за синтаксис и ищет баги)
index.html - входно html файл
package.json - вся информация про наше приложение
  scripts - команды чтобы быстрее их запустить через npm run "название команды"
  dependencies - основные зависимости(пакеты) нашего приложение без которых приложение не будет работать
  devDependencies - зависимости(пакеты) нужные только во время разработки приложения
README.md - документация для нашего приложения
vite.config.js - настройки vite
public - Файлы к которым можно получить доступ через статический URL: например, Если у вас есть сайт с URL -адресом www.example.com и изображение с именем IMG. PNG в public, тогда это изображение можно получить по адресу www.example.com/img.png.
src - логика и ядро приложения
  assets - Файлы будут объединены в другое место и не могут быть доступны через статический URL
  main.jsx - точка входа приложения
  App.jsx - компонент
